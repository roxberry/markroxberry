The Subpage Slider plugin runs through selected sub-pages like a slideshow.

== Description ==

The Subpage Slider plugin slides through sub-pages of any given parent page. The plugin user
has the option of order of the subpages and it allows for repeating certian pages over. Each
sub-page is refreshed using AJAX so it doesn't have to refresh the entire window. The user also
has the option to show what the first link will be. After the slidesohw it cycles back to the first
slide.


== Installation ==

1. Upload `subpage-slider` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the "Subpage Slider" into a sidebar, then edit the properties
4. For more information on using it within a page please goto http://cameronpreston.com/projects/plugins/subpage-slider
